# 🤖 Jarvis AI Assistant

A Python-based voice assistant that listens to commands and performs tasks like opening websites, playing music, fetching cricket scores, and more.

---

## 🚀 Features

* 🎤 Voice recognition
* 🔊 Text-to-speech responses
* 🌐 Open websites (Google, YouTube, LinkedIn, Instagram)
* 🎵 Play songs on Spotify
* 🔍 Search on YouTube
* 🏏 Live cricket scores
* ⏰ Current time
* 🌦️ Weather updates

---

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* Requests
* Webbrowser

---

## 📦 Installation

```bash
git clone https://github.com/your-username/jarvis-ai-assistant.git
cd jarvis-ai-assistant
pip install -r requirements.txt
```

---

## ▶️ Run

```bash
python main.py
```

---

## 🗣️ Example Commands

* "Open Google"
* "Play Believer"
* "Tell me score"
* "Tell me weather"

---

## ⚠️ Notes

* Works best on Windows
* Requires microphone & internet
